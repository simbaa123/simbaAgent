# README 更新要点（UI/交互优化）

> 用于在项目 README.md 中新增/更新的“本次改动”说明（不涉及技术栈变更）。

## 更新摘要
- Inbox：强化会话卡片信息层级与状态 Badge；补齐 loading/空/错态与重试入口。
- Console：对话区消息气泡与角色视觉区分；流式输出“正在生成”状态更清晰；输入区支持更友好的键盘交互（Enter 发送、Shift+Enter 换行）。
- 侧栏：订单信息与物流轨迹时间线卡片化，异常状态更易识别；未加载/无数据提示更一致。
- 可观测性：Plan/工具日志信息对齐与折叠策略优化，便于扫读；结果（导出/审计）卡片更突出并支持快速清除。
- 弹窗：need_choice/need_confirm 统一弹窗样式与按钮层级，减少误触与信息丢失。

## 不变项声明
- 不改变现有前后端接口、SSE 事件协议与 MCP 能力。
- 不引入新的前端框架/组件库；仅做样式规范化、布局与交互状态一致性优化。
